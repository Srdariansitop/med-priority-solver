# 🏥 Emergency Triage AI 

Breve descripción: Sistema inteligente de clasificación de pacientes en urgencias utilizando LLMs (Groq) y modelos locales (Transformers) para analizar síntomas, crónicos y recursos hospitalarios.

## 🚀 Requisitos previos

- Python 3.10 o superior instalado en tu sistema.
- Git (opcional, si clonas el repositorio).
- Una clave API de [Groq](https://console.groq.com/) (gratuita) para el análisis con LLM.
